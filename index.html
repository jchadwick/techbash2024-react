What is React?

  • JavaScript library for building user interfaces
  • Developed by Facebook in 2013
  • Manages the view layer in web (or native!) applications

React is efficient, declarative, and flexible

Key Features of React

  1. Component-Based
    Encapsulated components manage their own state and have well-defined contracts with inputs and outputs (aka events) 
    
                     <Title>Current Users</Title>

                     <UsersList 
                        users={selectedUsers} 
                        onUserSelected={handleUserSelected} 
                     />
    Potential for reuse (but that's not the point!)

  3. Virtual DOM

Key Features of React

  1. Component-Based
  2. JSX Syntax (JavaScript XML) allows you to mix HTML and JavaScript 
    
                        function UsersList() {
                           const users = [ { name: 'Alice' }, { name: 'Bob' } ];

                           return (
                              <ul>
                                 {users.map(user => (
                                    <li>{user.name}</li>
                                 ))}
                              </ul>
                           );
                        }
    Transpiled to regular JavaScript by tools like Babel
  3. Virtual DOM

Key Features of React

  1. Component-Based
  2. JSX Syntax
  3. Virtual DOM
    • In-memory copy of actual DOM
    • When the state changes, React:
      1. Re-renders affected components into the virtual DOM
      2. Compares the new virtual DOM with the previous one
      3. Applies only the differences to the real DOM
